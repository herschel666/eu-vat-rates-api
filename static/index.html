EU VAT Rates API

Current VAT rates fetched from ec.europa.eu.

Note: This project is work in progress. Expect the structure of the JSON response to change. 


curl https://eu-vat-rates-api.netlify.app/api | jq
    

curl https://eu-vat-rates-api.netlify.app/api/<country code> | jq
